### 消息队列

redis的lpush和brpop命令组合即可实现阻塞队列

生产者客户端使用lpush从列表左侧插入元素，多个消费者客户端使用brpop命令阻塞式的抢列表尾部的元素

多个客户端保证了消费的负载均衡和高可用性
